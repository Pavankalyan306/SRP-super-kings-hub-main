# 🏏 SRP Village Cricket

A dynamic cricket scoring and player management platform built using modern web technologies with AI-assisted development.

---

## 🚀 Project Overview 

**SRP Village Cricket** is a full-stack web application designed to manage and digitally track village-level cricket matches and tournaments.

The platform enables live scoring, player performance tracking, team management, and administrative control through a structured and user-friendly interface.

This project was developed using modern frontend technologies and AI-assisted development tools (Lovable AI) to accelerate UI design, layout generation, and feature structuring.

---

## 🎯 Key Features

- 🏏 Live Ball-by-Ball Match Scoring  
- 📊 Automated Player Statistics (Runs, Wickets, Strike Rate, Economy)  
- 🧑‍🤝‍🧑 Team & Player Management  
- 🏆 Match History & Tournament Records  
- 📈 High Score & Top Wicket Filters  
- 🔐 Secure Admin Dashboard  
- 📝 Dynamic About Page (Admin Editable)  
- 📂 Resume & Profile Upload Support  
- 📬 Contact Form System  
- 🌐 GitHub API Integration
